Mode

Online — two players

Use the same site on two devices (or two browser tabs). Player 1 creates the room and shares a code. Player 2 joins with that code. Then both lock a fighter; Player 1 starts the battle.

Player 1 — Host
  1. Tap Create room (you get a code).
  2. Send the code to Player 2 (message, voice, etc.).
  3. In the form below, enter your fighter and tap Lock in your character.
  4. When both players are locked, tap Begin battle.
Your room code

Tell Player 2 to enter this code under “Player 2 — Guest” (or in the join box if they still see it).
Player 2 — Guest
  1. Get the 5-digit code from Player 1.
  2. Type it in the box, then tap Join with code.
  3. Enter your fighter in the panel that appears, then Lock in character.
  4. Wait — Player 1 starts the match with Begin battle.

Your fighter (Player 2)

When the lobby says both fighters are locked, start the match for both players.
